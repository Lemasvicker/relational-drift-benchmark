# Security Policy

## Supported Versions
This repository primarily distributes datasets and documentation. If code is added in the future, security fixes will be issued through tagged releases.

## Reporting a Vulnerability
If you believe you’ve found a security or privacy issue (e.g., accidental inclusion of sensitive data),
please open a **private** report if the platform supports it, or contact the maintainer through the email listed
on the associated preprint/Zenodo record.

Please do not disclose sensitive issues publicly until they have been addressed.
