# Governance

## Maintainer
Victory Nlemadim

## Scope
RDE is an evaluation benchmark and labeling rubric for measuring relationship-effects in mental-health LLM outputs.
It is not a clinical product.

## Decision making
- Changes affecting categories, labeling schema, or contracts require a version bump.
- Breaking changes (schema changes) require a major version bump.

## Releases
Releases should be tagged and (ideally) archived to Zenodo for DOI versioning.
